# الصفحة الرئيسية – الثقافة السعودية

مشروع صفحة رئيسية عربية تعرض جوانب من الثقافة السعودية: التراث، الفنون، المطبخ، والمعرض.

## المحتوى
- `index.html`: الصفحة الرئيسية مع أقسام متعددة.
- `assets/styles.css`: الأنماط والتصميم المستجيب.
- `assets/script.js`: تفاعلات بسيطة (قائمة الجوال والتمرير السلس).
- `Fa.html`: صفحة سابقة موجودة بالمجلد (غير مستخدمة من الصفحة الجديدة).

## طريقة التشغيل محليًا
1. افتح المجلد في المتصفح مباشرة:
   - انقر مرتين على `index.html`، أو
   - اسحب الملف إلى نافذة المتصفح.
2. أو استخدم خادم محلي بسيط (اختياري):
   - Python: `python -m http.server 8000`
   - ثم افتح `http://localhost:8000/index.html`
